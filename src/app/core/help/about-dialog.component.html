About Dancing Clover

This application is a non-profit application to help people memorize Irish Set Dances.
It will be regularly updated to add more features and dances.

Enable offline mode for Android users:

You can add this website to your applications list by clicking on Add to Home screen in Chrome, this will transform the website into an application that can even work when there is no network. Give it a try!
